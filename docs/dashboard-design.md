# OpenCode Stats Dashboard — UI 设计规范

## 概述

OpenCode Stats Dashboard 是一个本地 Web 仪表板，用于展示 OpenCode 会话统计、已删除会话审计、工具调用详情。技术栈：Vue 3 + Vite + TypeScript，后端 Rust Salvo 边车提供 API。

---

## 整体布局

**导航模式**：顶部导航（水平导航栏）

**布局结构**：
```
┌─────────────────────────────────────────────────────┐
│ Logo  |  Overview  |  Sessions  |  Tool Calls  | ... │  ← 顶部导航
├─────────────────────────────────────────────────────┤
│                                                     │
│                    主内容区                          │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**实时状态栏**（嵌入顶部导航右侧）：
- SSE 连接状态（绿点/红点）
- 边车运行状态（绿点/红点）
- 审计级别标签（完整/受限）
- 隐私模式标签

### 1. Overview（概览页）

**布局**：宽屏充分利用，信息密度最大化

```
┌─────────────────────────────────────────────────────────────┐
│ [SSE ●] [边车 ●] [端口]              [审计: 完整] [隐私模式] │  ← 状态栏
├─────────────────────────────────────────────────────────────┤
│ ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐            │
│ │ 总会话  │ │ 已删除  │ │ 总代币  │ │ 总成本  │            │  ← 指标卡
│ │  128    │ │   23    │ │  1.2M   │ │ $12.34  │            │
│ │+12 本周 │ │ 18% 率  │ │入800K出 │ │今日$1.23│            │
│ └─────────┘ └─────────┘ └─────────┘ └─────────┘            │
├─────────────────────────────────────┬───────────────────────┤
│         使用趋势图                   │      按模型分布        │
│     （7天柱状图，会话+删除叠加）      │    （进度条占比）      │
│           [7天] [30天] [全部]        │    sonnet-4 62%       │
├───────────────┬───────────────┬─────┴───────────────────────┤
│  按项目分布   │  近期会话     │      工具调用 Top 5          │
│  （会话数排行）│ （时间线+标签）│     （使用次数排行）         │
└───────────────┴───────────────┴─────────────────────────────┘
```

**指标卡详情**：
| 指标 | 主数值 | 辅助信息 |
|---|---|---|
| 总会话 | 128 | 105 活跃 · 23 已删除 |
| 已删除 | 23 | 18% 删除率 |
| 总代币 | 1.2M | 输入 800K · 输出 400K |
| 总成本 | $12.34 | 今日 $1.23 · 本周 $4.56 |

**趋势图**：
- X 轴：7 天（Mon-Sun）
- Y 轴：会话数
- 双色柱状图：蓝色=会话数，红色=已删除数（叠加）
- 时间范围切换：7天 / 30天 / 全部

**底部三栏**：
- 按项目分布：项目路径 + 会话数，"查看全部 →"链接
- 近期会话：会话标题 + 时间 + 活跃/已删除标签
- 工具调用 Top 5：工具名 + 调用次数

### 2. Sessions（会话审计表）

**布局**：过滤器 + 表格 + 分页 + 批量操作

```
┌─────────────────────────────────────────────────────────────┐
│ [搜索框__________] [状态▼] [模型▼] [项目▼] [日期] 至 [日期] [重置] │
├─────────────────────────────────────────────────────────────┤
│ 共 128 会话 | 105 活跃 | 23 已删除 | 已选 0 条               │
├─────────────────────────────────────────────────────────────┤
│ ☐ │ 会话ID ↕ │ 项目路径 ↕ │ 模型 ↕ │ 代币 ↕ │ 成本 ↕ │ 状态 ↕ │ 首次事件 ↕ │ 末次事件 ↕ │
├───┼──────────┼──────────┼──────┼──────┼──────┼──────┼─────────┼──────────┤
│ ☐ │ ses_abc  │ ~/proj   │ son  │12,345│ $0.12│ 活跃 │ 06-01   │ 2分前    │
│ ☐ │ ses_def  │ ~/dash   │ son  │ 8,901│ $0.09│ 活跃 │ 06-01   │ 1时前    │
│ ☐ │ ses_ghi  │ ~/api    │ gpt  │15,678│ $0.18│ 删除 │ 05-31   │ 昨天     │ ← 浅红背景
└───┴──────────┴──────────┴──────┴──────┴──────┴──────┴─────────┴──────────┘
│ 显示 1-50 / 共 128 条                               [上一页] [1] [2] [3] [下一页] │
├─────────────────────────────────────────────────────────────┤
│ 批量操作：[导出选中] [导出全部] [删除选中]                     │
└─────────────────────────────────────────────────────────────┘
```

**过滤器**：
- 搜索框：搜索会话 ID、项目路径
- 状态下拉：全部状态 / 活跃 / 已删除
- 模型下拉：全部模型 / sonnet-4 / gpt-4o / ...
- 项目下拉：全部项目 / ~/projects/...
- 日期范围：开始日期 - 结束日期
- 重置按钮

**表格列**：
| 列 | 对齐 | 说明 |
|---|---|---|
| 复选框 | 居中 | 批量选择 |
| 会话 ID | 左对齐 | 等宽字体 `ses_xxx` |
| 项目路径 | 左对齐 | 截断显示 |
| 模型 | 左对齐 | — |
| 代币 | 右对齐 | 数字格式化（千分位） |
| 成本 | 右对齐 | $X.XX 格式 |
| 状态 | 居中 | 标签：活跃=绿、已删除=红 |
| 首次事件 | 左对齐 | YYYY-MM-DD HH:mm |
| 末次事件 | 左对齐 | 相对时间（2分钟前） |

**视觉区分**：
- 已删除会话行：浅红背景 `rgba(239,68,68,0.05)`
- 排序指示器：↕ 箭头

**分页**：每页 50 条，页码按钮

**批量操作**：导出选中、导出全部、删除选中

### 3. Tool Calls（工具调用审计表）

**布局**：受限审计警告 + 过滤器 + 表格 + 分页

```
┌─────────────────────────────────────────────────────────────┐
│ ⚠️ 受限审计模式                                              │
│ OpenCode 事件表面未暴露工具调用生命周期，仅显示会话级别统计。  │
├─────────────────────────────────────────────────────────────┤
│ [搜索框__________] [工具▼] [状态▼] [会话▼] [日期] [重置]       │
├─────────────────────────────────────────────────────────────┤
│ 共 2,456 次调用 | 2,102 成功 | 312 进行中 | 42 失败            │
├─────────────────────────────────────────────────────────────┤
│ 工具名 ↕ │ 会话ID ↕ │ 状态 ↕ │ 模型 ↕ │ 代币 ↕ │ 成本 ↕ │ 开始 ↕ │ 完成 ↕ │ 摘要    │
├──────────┼──────────┼──────┼──────┼──────┼──────┼──────┼──────┼─────────┤
│ read_file│ ses_abc  │ 完成 │ son  │1,234 │$0.01 │10:30 │10:30 │ 读取配置│
│ bash     │ ses_abc  │ 完成 │ son  │3,456 │$0.03 │10:30 │10:30 │ npm安装 │
│ write_f  │ ses_def  │ 进行 │ son  │  —   │  —   │10:31 │  —   │ 写入文件│

└──────────┴──────────┴──────┴──────┴──────┴──────┴──────┴──────┴─────────┘
│ 显示 1-50 / 共 2,456 条                             [上一页] [1] [2] ... [50] [下一页] │

```

**受限审计警告**（黄色警告条）：
- 条件：Phase 0 验证结果为 `fallback-session-level` 时显示
- 内容：说明"仅显示会话级别统计"，链接到验证报告

**过滤器**：
- 搜索框：搜索工具名、会话 ID
- 工具下拉：全部工具 / read_file / write_file / bash / grep / edit
- 状态下拉：全部状态 / 进行中 / 已完成 / 失败
- 会话下拉：全部会话 / ses_xxx
- 日期选择器

**表格列**：
| 列 | 对齐 | 说明 |
|---|---|---|
| 工具名称 | 左对齐 | 等宽字体 |
| 会话 ID | 左对齐 | 等宽字体 `ses_xxx` |
| 状态 | 居中 | 完成=绿、进行中=蓝、失败=红 |
| 模型 | 左对齐 | — |
| 代币 | 右对齐 | 进行中显示 "—" |
| 成本 | 右对齐 | 进行中显示 "—" |
| 开始时间 | 左对齐 | HH:mm:ss |
| 完成时间 | 左对齐 | 进行中显示 "—" |
| 摘要 | 左对齐 | 截断显示，tooltip 展开 |

**注意**：无 Input/Output 列（隐私默认值）

### 4. Exports（导出与清理）

**布局**：导出四宫格 + 清理区域

```
┌─────────────────────────────────────────────────────────────┐
│ 数据导出                                                     │
├───────────────────────┬─────────────────────────────────────┤
│ ┌───────────────────┐ │ ┌───────────────────┐               │
│ │ 会话数据     128条 │ │ │ 工具调用数据 2456条│               │
│ │ 字段：ID、项目...  │ │ │ 字段：工具、会话...│               │
│ │ [CSV] [JSON]      │ │ │ [CSV] [JSON]      │               │
│ └───────────────────┘ │ └───────────────────┘               │
│ ┌───────────────────┐ │ ┌───────────────────┐               │
│ │ 概览统计           │ │ │ 全量导出           │               │
│ │ 包含：总计、分布   │ │ │ 打包为 ZIP         │               │
│ │ [CSV] [JSON]      │ │ │ [导出全部 ZIP]     │               │
│ └───────────────────┘ │ └───────────────────┘               │
├───────────────────────┴─────────────────────────────────────┤
│ 数据清理 ⚠️                                                  │
│ 清理操作不可逆，建议先导出数据。                               │
├───────────────────────┬─────────────────────────────────────┤
│ ┌───────────────────┐ │ ┌───────────────────┐               │
│ │ 清理已删除会话记录 │ │ │ 清理全部数据       │               │
│ │ 仅删除"已删除"的   │ │ │ 删除所有记录       │               │
│ │ 审计记录（23条）   │ │ │ （不可恢复）       │               │
│ │ [清理已删除记录]   │ │ │ [清理全部数据]     │               │
│ └───────────────────┘ │ └───────────────────┘               │
└───────────────────────┴─────────────────────────────────────┘
```

**导出卡片**：
| 导出项 | 内容 | 按钮 |
|---|---|---|
| 会话数据 | 会话 ID、项目路径、模型、代币、成本、状态、时间 | CSV / JSON |
| 工具调用数据 | 工具名、会话 ID、状态、模型、代币、成本、时间、摘要 | CSV / JSON |
| 概览统计 | 汇总统计和维度分布 | CSV / JSON |
| 全量导出 | 打包为 ZIP | 导出全部 ZIP |

**清理区域**（红色警告样式）：
| 清理项 | 说明 | 按钮 |
|---|---|---|
| 清理已删除会话记录 | 仅删除标记为"已删除"的审计记录 | 清理已删除记录 |
| 清理全部数据 | 删除所有会话、工具调用、统计记录（不可恢复） | 清理全部数据（红色） |

**确认对话框**：
- 弹窗居中
- 标题：确认清理
- 说明：此操作将删除 N 条记录，数据一旦删除无法恢复
- 按钮：取消 + 确认删除（红色）

## 设计 Token

**颜色**（建议深色主题）：
- 主色：`--primary`（蓝色）
- 成功：`--success`（绿色）
- 危险：`--danger`（红色）
- 警告：`--warning`（黄色）
- 背景：`--bg`
- 表面：`--surface`
- 边框：`--border`
- 文本：`--text`
- 次要文本：`--text-muted`

**间距**：
- 卡片间距：12px
- 内边距：14-16px
- 表格行高：8px 12px

**圆角**：
- 卡片：6px
- 按钮：4px
- 标签：10px

**字号**：
- 标题：14px
- 正文：12px
- 辅助：11px
- 标签：10px

## 组件清单

| 组件 | 用途 |
